# Curso Git e Github para iniciantes

## Aula 009
- Untracked
	- Arquivo que o git ainda não viu;

- Unmodified
	- Arquivo que não foi modificado;

- Modified
	- Arquivo que foi modificado;

- Staged
	- Arquivo que serão enviados para o git no commit.

>
> ### Comandos mostrados:
> > *git status*
> > *git add nome_do_arquivo*
> > *git commit*
